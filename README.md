## üöÄ Features

*   **Real-time Analysis**: Paste any news article to instantly check its authenticity.
*   **Confidence Score**: Displays the model's confidence probability.
*   **Simple & Clean UI**: Built with Streamlit for a responsive user experience.
*   **Machine Learning**: Uses TF-IDF Vectorization and Logistic Regression trained on news datasets.

## üõ†Ô∏è Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/arvinth777/fake-news-detection.git
    cd fake-news-detection
    ```

2.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Generate Model Files:**
    The app requires `model.pkl` and `vectorizer.pkl`. You can generate these by running the included script.
    *   If you have the original `True.csv` and `Fake.csv` datasets, place them in the root directory to train on real data.
    *   If not, the script will create a demo model using dummy data.
    ```bash
    python generate_models.py
    ```

4.  **Run the App:**
    ```bash
    streamlit run streamlit_app.py
    ```

## üì¶ Deployment on Streamlit Cloud

This app is ready to be deployed instantly on Streamlit Cloud.

1.  **Push to GitHub**: Ensure this code is pushed to your GitHub repository.
2.  **Log in to Streamlit**: Go to [share.streamlit.io](https://share.streamlit.io/) and log in with GitHub.
3.  **Deploy**:
    *   Click "New app".
    *   Select your repository (`arvinth777/fake-news-detection`).
    *   Select the branch (e.g., `main`).
    *   Set **Main file path** to `streamlit_app.py`.
    *   Click **Deploy!**

Once deployed, copy the URL and update the "Streamlit App" badge link at the top of this README.
